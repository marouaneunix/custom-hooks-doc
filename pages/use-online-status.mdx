# use-oneline-status hook

## Get Started

import { Callout } from 'nextra-theme-docs'


## Description

This hook is used to get the online status of the user. 
It returns an enum value indicating the user's online status. 
The possible values are 'ONLINE' and 'OFFLINE'.

## Example

You just need to import `useOnlineStatus` from `@melmerrouni/custom-hooks`:

```tsx filename="demo.tsx" copy

import {useOnlineStatus} from '@melmerrouni/custom-hooks';

const Target = () => {
  const {status} = useOnlineStatus();

    return (
        <>
            <div className={status} data-testid="target">{status}</div>
        </>
    )
};
```
